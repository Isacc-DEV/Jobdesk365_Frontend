# Jobdesk365 Frontend

## Environment
Use Vite env variables from `.env` or `.env.production`.

## Minimum Production Env Checklist
- `VITE_API_BASE_URL` (required)
- `VITE_BACKEND_ORIGIN` (optional, defaults to API base origin)
- `VITE_AI_SERVICE_URL` (optional, defaults to `${VITE_API_BASE_URL}/ai/resume-tailor`)
